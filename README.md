# Bootcamp Luizalabs - Backend Python

Este repositório contém os projetos e atividades desenvolvidos durante o **Bootcamp Luizalabs em parceria com a DIO**, com foco em **Backend Python**.

## 🚀 Objetivo
Aprofundar conhecimentos em Python para backend, explorando desde fundamentos da linguagem até práticas modernas de desenvolvimento de APIs e uso de Inteligência Artificial para acelerar o processo de criação.

## 📚 Conteúdo do Bootcamp

- **Onboarding & Fundamentos de Python e Lógica de Programação**  
  Introdução ao ecossistema Python, sintaxe básica, estruturas de dados e lógica de programação.

- **Python Orientado a Objetos na Prática**  
  Aplicação dos conceitos de classes, herança, polimorfismo e boas práticas de programação orientada a objetos.

- **APIs Modernas com FastAPI do Zero ao Deploy**  
  Criação de APIs REST modernas utilizando FastAPI, com foco em performance, documentação automática e deploy em ambientes reais.

- **IA para Acelerar o Desenvolvimento**  
  Uso de ferramentas de Inteligência Artificial para aumentar produtividade, gerar código e otimizar processos de desenvolvimento.

## 🛠️ Tecnologias Utilizadas
- Python 3.x
- FastAPI
- Git & GitHub
- Ferramentas de IA aplicadas ao desenvolvimento

## 📂 Estrutura do Repositório
- `fundamentos/` → Exercícios e exemplos de lógica e sintaxe básica  
- `poo/` → Projetos aplicando conceitos de orientação a objetos  
- `fastapi/` → APIs desenvolvidas com FastAPI  
- `ia/` → Experimentos e exemplos de uso de IA no desenvolvimento
